dir - mostra a lista de diretórios que tem na pasta situada
cd / leva pra base do diretório C
cd .. pra voltar um nível
cd + nome da pasta para ir para a pasta
cls - limpar tela
tab - auto completar
mkdir + nome escolhido -  criar uma pasta
echo + escrita - Retorna o que você escrever
echo + escrita + > + nome de arquivo.txt - cria um arquivo e coloca dentro dele o que você escreveu
dell + nome da pasta - deleta tudo que tem dentro da pasta, mas não deleta a pasta
setas pra cima e baixo procuram os comandos na hordem que foram executados
rmdir + nome da pasta + /S + /Q - Deleta a pasta
rm dir
/S - Remove todas as pastas e arquivos da pasta especificada, além dela mesma. Utilizado para remover uma árvore de pastas.
/Q - Modo silencioso. Não pede confirmação para remover a árvore de pastas ao se passar o parâmetro /S
Comando + /? - Informa o que o comando faz
Botão direito do mouse + Git bash here - abre o git na pasta que você escolheu
openssl sha1 texto.txt - cria uma chave de 40 caracteres
Tree armazena blob que tem um sha1
arvore tem sha1 e blob também
Chave SSH, conectar minha máquina
$ eval $(ssh-agent -s)
Ctrl+l - Limpa a tela no bash
ls -a (mostra arquivos ocultos)
